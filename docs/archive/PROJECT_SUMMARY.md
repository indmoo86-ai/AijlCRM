## 🚀 快速启动

### 最简单的方式（Docker）

```bash
cd CRM/deploy
cp .env.example .env
docker-compose up -d
```

访问 http://localhost

### 本地开发

```bash
# 后端
cd CRM/backend
npm install
cp .env.example .env
npm run dev

# 前端
cd CRM/frontend
npm install
npm run dev
```

访问 http://localhost:5173

2. 修改 deploy/.env 配置
3. 执行 docker-compose up -d
4. 配置Nginx和SSL证书
